# 复杂诉讼项目协作演示

> 演示 Git 驱动的律师团队 AI 协作方式

## 场景说明

这是一个关于**软件技术合作纠纷**的复杂诉讼项目。代理原告方，主张被告违反技术合作协议。

本案的特点：
- 多方向研究任务并行推进
- 大量技术文档需要梳理
- 团队多人协作，边界清晰
- AI agent 可参与各分支工作
