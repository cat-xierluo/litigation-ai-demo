# 证据 A：TD-8800 Pro 技术白皮书

## 产品概述

TD-8800 Pro 是我方委托深圳腾达电子有限公司生产的无线路由器产品，采用自有协议栈实现数据传输。

## 核心技术实现

### 数据传输流程

1. **数据接收**：从 WAN 口接收以太网帧
2. **协议解析**：解析 TCP/IP 协议栈
3. **数据处理**：
   - 根据业务类型（视频/语音/数据）采用不同队列
   - 采用不等长分段策略（128-1500 字节）
4. **无线发射**：固定 QPSK 调制，固定转发速率

### 关键技术参数

| 参数 | 数值 |
|------|------|
| 最大传输速率 | 1200 Mbps |
| 调制方式 | QPSK（固定） |
| 分段策略 | 可变长度 |
| 错误控制 | ARQ |

### 与涉案专利的关键差异

涉案专利采用自适应调制，而我方产品采用固定 QPSK 调制。这导致：
- 我方产品无法根据信道状态动态调整传输效率
- 技术实现成本更低，但性能也相应降低
